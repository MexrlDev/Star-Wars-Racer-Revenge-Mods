# Explaining how things work
* First of all you need the source code (do a fork of the [nes emu](https://github.com/egycnq/EmuC0re)) secondly use the YML script I included to build the bin file, it comes with elf but you’ll need the bin only. After building it use the bin-to-lua.py script to cover the bin to lua hex string. You’ll need to put the bin file in a folder called bin next to the python file!, then the python will create the output of the file as a hex string you can paste in nes.lua to replace the original string. And there you have it.
